# What is Cohort Analysis?
Cohort analysis is a way of grouping users by their sign-up date to see how their behavior changes over time. Instead of looking at everyone as one big group, it helps us track how specific groups (cohorts) stay engaged or drop off, so we can build a better product.

- Spot Churn: Identify exactly when users stop using the product.

- Measure Growth: Understand if new product updates are actually improving user retention over time.
